# 개요

recommended-n-search 페이지에서 제공되는 검색기능을 위한 TypeScript 함수. 일단은 단순 keyword 검색 기능만 구현.


# Pseudo code

## input

Props {
    query: string,
    contents: list[dict{
        keyword: list[string],
        title: string,
        description: string,
    }],
}

## function

def keyword_search(query: string, contents) :
    return [c for c in contents if query in (c.keyword + c.title + c.description)]
