# 개요

recommended-n-search 페이지에서 제공되는 검색기능을 위한 TypeScript 함수. Pure function으로 구현하여 재사용성과 테스트 용이성 확보.

# Design Pattern
- **Pure Function**: 부수효과 없이 입력에 대한 출력만 반환.
- **Filter Pattern**: 조건에 맞는 데이터만 필터링.

# Input / Output

## Input
Props {
    query: string,
    contents: list[dict{
        keyword: list[string],
        title: string,
        description: string,
        favicon: string,
        url: string,
    }],
}

## Output
ReturnValue {
    filteredContents: list[dict{
        keyword: list[string],
        title: string,
        description: string,
        favicon: string,
        url: string,
    }]
}

# Pseudo code

## function

def keyword_search(query: string, contents: list[dict]) -> list[dict] :
    // 대소문자 구분 없이 검색
    query_lower = query.toLowerCase()
    
    return [c for c in contents if (
        query_lower in c.title.toLowerCase() or
        query_lower in c.description.toLowerCase() or
        any(query_lower in k.toLowerCase() for k in c.keyword)
    )]
