# 개요

추천페이지와 동일한 계층에 삽입되는 Sponsor 광고 컴포넌트. 

# Design Pattern
- **Placeholder Pattern**: 향후 실제 광고 컴포넌트로 교체 가능한 구조.
- **Presentational Component**: 단순 표시 역할.

# Input / Output

## Input
Props {
    sponsorUrl: string = "https://github.com/deliciouscat",
    sponsorTitle: string = "Sponsor",
    sponsorDescription: string = "Support this project"
}

## Output
Output {
    onSponsorClick: (url: string) => void
}

# Pseudo code

import color_template.css

Box(
    element: 
        Placeholder(    // 사각형에 사선
            width: 100%,
            height: 100px,
            onClickUrl: Props.sponsorUrl
        ),
    fill: {
        borderColor: color_template.grey-lv2,
        borderWidth: 1px,
        borderStyle: "dashed",
        hover: color_template.grey-lv1,
    },
    onPress: () => {
        OpenNewTab(Props.sponsorUrl)
    }
)
